# Crypto Ticker App (Ionic + Vue)

Aplikasi mobile sederhana untuk menampilkan daftar cryptocurrency
menggunakan API Coinlore.

## Fitur
- Fetch data real-time dari API
- Komponen CryptoCard terpisah
- Struktur folder profesional (services, components, views)
- Loading indicator
- UI modern pastel green
- Responsive layout

## Technologies
- Ionic Vue
- Vue 3 Composition API
- Vite
- TypeScript
- Coinlore API

## Cara Menjalankan
npm install
ionic serve
